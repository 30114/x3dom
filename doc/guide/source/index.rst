Welcome to X3DOM
================

.. .. image:: _static/x3dom-logo-small.png
..    :alt: X3DOM: Declarative 3D for the Web
..    :class: floatinglogo

Welcome to X3DOM's documentation. This documentation is divided into
different parts. We recommend that you get started with
:ref:`installation` and then head over to the :ref:`quickstart`.
Besides the quickstart there is also a more detailed :ref:`tutorial` that
shows how to create a complete (albeit small) applications with X3DOM.  If
you'd rather dive into the internals of X3DOM, check out
the :ref:`api` documentation.
